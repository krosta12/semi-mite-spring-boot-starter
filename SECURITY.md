# Security Policy

## Supported Versions
Security updates are currently provided only for the latest development version of the project.

| Version | Supported |
| ------- | --------- |
| latest  | Yes       |

## Reporting a Vulnerability
If you discover a potential security vulnerability (such as critical native memory leaks, out-of-bounds access risks, or unsafe exposures via the Panama API), please do not open a public Issue.

Instead, please report it privately by opening a Security Advisory directly through the GitHub repository settings, or contact the maintainer directly. This allows us to investigate and resolve the issue before public disclosure.
